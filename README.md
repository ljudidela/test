# Sochi Webcams Map

Интерактивная карта вебкамер города Сочи.

## Функционал
- Интерактивная карта (Leaflet)
- Список камер с превью
- Адаптивный интерфейс (Mobile/Desktop)
- Модальное окно просмотра видео

## Запуск

```bash
npm install
npm run dev
```

## Стек
- React + TypeScript + Vite
- Tailwind CSS
- React Leaflet
